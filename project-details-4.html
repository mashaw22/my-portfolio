What is Galleria?

Galleria is a responsive gallery of historical paintings. Made with React and Gatsby.js, this site allows for the user to click each painting to display a dynamically-rendered details page and paginate forwards and backwards through other paintings.

Achievements

This was the first time I went outside of my frontend comfort zone and dove into backend (albeit just a little). This project piqued my interest into learning the full backend process, specifically of Node.js.

Learning Curves

For this project, I had to learn how to fetch data from GraphQL. The GraphiQL IDE made it easy to select the specific data needed, whether I needed to return data for all of the paintings or a single selection.

Pagination was a huge stepping stone for me! After setting up a Galleria Directions component, I linked the previous and next pages based on the currently-selected page. It took some getting used to since I used Node.js to pull the functionality of pagination.

Future Enhancements

I was unable to implement the slideshow feature. I tried using a timed pagination through all of the elements of galleria array, but I continued getting errors about how the site was unable to read the .length() property.

Further, I translated the data.json to markdown to test out GraphQL in the way that I learned it. However, this app could use a backend database instead to store the data, and a user could input a new painting with its details on the frontend.
