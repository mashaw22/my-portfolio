What is the Invoice Creator?

I created this Invoice Creator as a part of Scrimba's Solo Project. Working from a Figma file, I coded the HTML and CSS using some custom properties.

When a user clicks on a service button, such as "Wash Car: $10," a new div is created containing HTML listing the service and cost. The button becomes disabled so as to prevent the user from adding the service again.

Next Steps?

This application can be adjusted with some media queries so that it is responsive at smaller screens. Currently, the service buttons get too close to each other on small screens. I would also like to add a small "Remove" button next to each added service and improve the class naming convention to BEM.

Further, I am still working on making the JavaScript DRY-er by pulling the function out of the servicesArray.map(). A real challenge would be to give the user the ability to input their email so that a real invoice can be sent when the "Send" button is clicked.
